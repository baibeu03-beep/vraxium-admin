## 7. B7 apply 가능 여부 (#7 결론)

**조건부 가능** — 차단/선행 항목:

1. **실사용자 안전 확인됨**: enforced 행 = 전원 테스터 → threshold 백필이 실사용자 표시를 바꾸지 않음 (uws 불변 + read-time 게이트 미강제). `syncExperienceGrowthWeekStatuses` 도 레거시 주차 update 금지(관찰만) — uws 소급 write 경로 없음 확인.
2. **테스터 358주차 flip**: 수용(시연 데이터 변형) 또는 threshold 37 기준 재시드 — 운영 결정 필요.
3. **apply 직후 snapshot 명시 재계산**: checks_migrated=true 보유 사용자(현재 90명) — checkGate.required 가 snapshot 에 구워져 있어 재계산 전까지 '내용 stale'.
4. **휴식 충돌 7건 수동 확정** (B7 기존 게이트 — 본 감사와 독립, 감사 25주와 겹치지 않음).
5. **(신규) 이관 파이프라인 §12-①-3 수정**: Vraxium-native 주차 덮어쓰기 금지 (Part C 결함) — B7 자체가 아니라 이관(dry-run 1명) 전 반영.

뒤집힘 정책(§5-①)은 **이관 시점** 결정 사항 — B7 apply(주차 기준값 백필)의 차단 요건은 아님 (실사용자는 미이관 상태라 게이트 비활성).
